# Sepsis Prediction Using ICU Data

## Overview
Predicts sepsis onset in ICU patients using machine learning classification
models trained on clinical ICU data. Early sepsis detection can directly
reduce patient mortality in hospital settings.

## Tools & Technologies
- Python, Pandas, NumPy
- Scikit-Learn (Logistic Regression, Decision Tree, Random Forest)
- Jupyter Notebook

## Approach
1. Data cleaning and preprocessing of ICU clinical records
2. Feature engineering on vital signs and lab values
3. Model training and cross-validation
4. Performance evaluation using accuracy, precision, recall, AUC-ROC

## Key Outcome
Built a classification model to flag high-risk sepsis patients early,
supporting clinical decision-making in ICU environments.

Healthcare Analytics · Machine Learning · Python · Clinical Data
